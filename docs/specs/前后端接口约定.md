# 前后端接口约定 v0.1

> 产品经理（窗口2）草拟 | 待后端（窗口3）确认 | 更新日期：2026-05-07

---

## 约定说明

- 所有 API 前缀：`/api/v1`
- 数据格式：JSON，编码 UTF-8
- 时间格式：ISO 8601 (`2024-01-15T00:00:00`)
- 金额单位：人民币元，保留2位小数
- 分页参数：`page`（从1开始）、`page_size`（默认50）
- 响应结构见"通用响应格式"章节
- ✅ 标记 = 与 `backend/models/` 中已有模型对齐

---

## 通用响应格式

```json
{
  "code": 0,
  "message": "ok",
  "data": {...}
}
```

错误响应：

```json
{
  "code": 40001,
  "message": "股票代码不存在",
  "data": null
}
```

错误码前缀：
- `400xx`：请求参数错误
- `500xx`：服务端异常
- `600xx`：数据源异常

## 1. 自选股 API

### 1.1 搜索股票

```text
GET /api/v1/stocks/search?keyword=万化&limit=10
```

**请求参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| keyword | string | 是 | 股票代码或中文名称关键字，支持中文模糊匹配，如"万化"匹配"万华化学" |
| limit | int | 否 | 返回条数，默认10，最大50 |

**响应 data**：搜索候选列表

```json
{
  "total": 1,
  "items": [
    {
      "code": "600309",
      "name": "万华化学",
      "exchange": "SH",
      "market_label": "沪A",
      "industry": null,
      "list_date": null,
      "is_watchlist": false
    }
  ]
}
```

说明：搜索框只支持代码和中文名称输入，不把拼音首字母作为主要输入方式。

---

### 1.2 获取自选股列表

```text
GET /api/v1/watchlist?tag=蓝筹&sort=change_pct&page=1&page_size=50
```

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| tag | string | 否 | 按分组筛选 |
| sort | string | 否 | 排序字段：code/name/change_pct，默认 `code` |
| page | int | 否 | 页码 |
| page_size | int | 否 | 每页条数 |

**响应 data**：`StockList` + 行情快照合并

```json
{
  "total": 30,
  "items": [
    {
      "code": "000001",
      "name": "平安银行",
      "exchange": "SZ",
      "industry": "银行",
      "is_watchlist": true,
      "tags": ["蓝筹"],
      "current": 10.52,
      "change": 0.12,
      "change_pct": 1.15
    }
  ]
}
```

## 2. 行情 API

### 2.1 K线数据

```text
GET /api/v1/market/kline/{code}?frequency=D&start=2024-01-01&end=2024-12-31
```

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| frequency | string | 否 | D/W/M，默认 D |
| start | date | 是 | 起始日期 |
| end | date | 是 | 截止日期 |

**响应 data** ✅ `KlineData[]`

```json
{
  "data": [
    {
      "code": "000001",
      "date": "2024-01-02T00:00:00",
      "open": 9.85,
      "high": 10.05,
      "low": 9.80,
      "close": 9.98,
      "volume": 12345678,
      "amount": 123456789.00,
      "frequency": "D"
    }
  ]
}
```

---

### 2.2 实时行情快照

```text
GET /api/v1/market/quote/{code}
```

**响应 data** ✅ `RealtimeQuote`

```json
{
  "data": {
    "code": "000001",
    "name": "平安银行",
    "current": 10.52,
    "change": 0.12,
    "change_pct": 1.15,
```

## 3. 策略回测 API

### 3.1 策略配置 CRUD

```text
GET    /api/v1/strategies           # 列表
POST   /api/v1/strategies           # 创建
GET    /api/v1/strategies/{id}      # 详情
PUT    /api/v1/strategies/{id}      # 更新
DELETE /api/v1/strategies/{id}      # 删除
```

**创建/更新请求体** ✅ `StrategyConfig`

```json
{
  "name": "双均线交叉-日线",
  "description": "5日线上穿20日线买入，下穿卖出",
  "params": {
    "fast": 5,
    "slow": 20
  }
}
```

当前 MVP 已内置的 `macd_daily` 预设策略为：MACD 绿柱连续缩短 3 天买入，MACD 红柱连续缩短 3 天卖出。

---

### 3.2 启动回测

```text
POST /api/v1/backtest
```

**请求体**

```json
{
  "strategy_id": "uuid-of-strategy",
  "stocks": ["000001", "000002"],
  "start_date": "2024-01-01",
  "end_date": "2024-12-31",
  "initial_capital": 100000.00,
  "commission_rate": 0.0003
}
```

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| strategy_id | string | 是 | 策略ID |
| stocks | string[] | 是 | 股票池，为空则用全部自选股；前端可通过股票搜索框选出代码后提交 |
| start_date | date | 是 | |
| end_date | date | 是 | |
| initial_capital | float | 是 | 初始资金，>= 10000 |
| commission_rate | float | 否 | 佣金费率，默认 0.0003 |

**响应**

```json
{
  "data": {
    "backtest_id": "uuid-of-backtest",
    "status": "running"
  }
}
```

### 4.5 生成交易信号

```text
POST /api/v1/paper/signals/generate?account_id=1&scope=holdings
```

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| account_id | int | 否 | 关联模拟账户 |
| scope | string | 否 | `holdings`=观察股/持仓（默认），`watchlist`=自选股 |
| codes | string | 否 | 逗号分隔股票代码；传入后按手动股票分析 |
| limit | int | 否 | 自选股分析上限，默认20，最大100 |

**响应 data**

```json
{
  "scope": "holdings",
  "total": 1,
  "buy_signals": [],
  "sell_signals": [
    {
      "id": 12,
      "code": "000001",
      "name": "平安银行",
      "signal_type": "SELL",
      "confidence": 0.68,
      "reason": "MACD红柱缩短",
      "close_price": 10.55,
      "status": "pending"
    }
  ],
  "all": [
    {
      "id": 12,
      "code": "000001",
      "name": "平安银行",
      "signal_type": "SELL",
      "confidence": 0.68,
      "reason": "MACD红柱缩短",
      "close_price": 10.55,
      "status": "pending"
    }
  ]
}
```

说明：默认只分析观察股（当前账户持仓），不会自动扫全部自选股或全市场；额外股票由前端搜索后通过 `codes` 手动传入。

## 附录：前后端数据结构映射

| 前端概念 | 后端 Pydantic 模型 | 文件 |
|----------|-------------------|------|
| 股票搜索结果 | `StockList` → `StockDetail[]` | `backend/models/stock.py` |
| K线数据 | `KlineData` | `backend/models/market.py` |
| 实时行情 | `RealtimeQuote` | `backend/models/market.py` |
| 策略配置 | `StrategyConfig` | `backend/models/strategy.py` |
| 策略信号 | `StrategySignal` | `backend/models/strategy.py` |

新增模型（需后端定义）：
- `BacktestRequest` / `BacktestResult` / `BacktestTrade`
- `PaperAccount` / `PaperPosition` / `PaperTrade`
- `IndicatorData`（MA/MACD/RSI多周期计算结果）
- `SignalResult`（信号输出：BUY/SELL/HOLD + 置信度+各周期明细）
